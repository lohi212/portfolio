UI Builder

UI Builder enables developers to author search UIs with no-code, extend them with low-code with a cloud IDE and publish search UIs at the global edge on your domain.

  • It is used to create a full-stack search UIS using JavaScript.
  • Templates: A set of search UIs were provided to user as templates and user can select the desired layout. The search app can be built either on ReactJS or VueJS.
  • UI Components: UI builder allows adding all the major components of the UI.
  • Authentication: User can configure end-user authentication with Username / Password and OAuth with Google. When configuring username/password authentication, you can optionally choose to enable user sign-ups or manage your end-users with the User Management Portal from within the dashboard.
  • Domain: You can publish the search UI to your own custom domain.
  • Theme: You can theme the search UI with your company logo, configure look and feel for the UI.
  • Settings: The behavior for search, filters, and results can be configured.
  • Built complex and nested forms and architected the state management for these form data.
  • Saving and retrieving the complex form data during re renders was a challenge.
  • Used Redux-toolkit to overcome the state updates and enable smooth re renders.
